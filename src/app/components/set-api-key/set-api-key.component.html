Mashape API Key

To get any of the demo examples to work, you enter a valid Mashape API key for an account that's subscribed to one of the GeoDB API plans.

Entering a wrong key or one with no plan subscription will not cause an error here. But in this case when trying to run the demos, you will get HTTP 403 Forbidden errors in the console log.

  1. Create an account on RapidAPI. (Rapid may ask for credit-card info. As long as you stay within the free usage limits of the Basic plan, you will not be charged.)
  2. Select a GeoDB plan.
